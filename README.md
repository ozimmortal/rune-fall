# RUNE FALL

Runefall is a 2D action-adventure fantasy game that follows the emotional and heroic journey of Aren, a devoted father whose daughter, Lina, is kidnapped by a powerful witch seeking to use the child’s hidden magical potential for her own dark rituals. The story unfolds across a mystical world once governed by ancient runes, sources of immense power that have now fallen into corruption and decay.
